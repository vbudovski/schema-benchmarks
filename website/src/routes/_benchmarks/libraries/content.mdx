Individual packages benchmarked by this site.

Sometimes one package will be separated into more than one set of benchmarks, for example if it has multiple ways of using it (e.g. [`zod`](https://zod.dev/packages/zod) and [`zod/mini`](https://zod.dev/packages/mini)), or there are multiple versions of note (such as [`effect`](https://effect.dev) and its [beta](https://effect.website/blog/releases/effect/40-beta/)).
